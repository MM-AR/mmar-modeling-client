User Info:

You are in Selection Mode:

You are in selection mode. This means you can drag around instances in the scene and resize them. Click on an instance to select it.

  • Click on an object enables the dragging mode.
  • Right-click on an object enables the scaling mode.
  • Middle-click on an object enables the rotation mode. (only if in 3D view mode)
  • When an object is selected you can drag or scale it by using the visualized UI elements.
  • Use the arrow keys when an object is selected to shift it to a selected direction.
  • To leave the selection mode you have to click or right-click on an empty area of the canvas.

You are in Drawing Mode:

This means you selected a Class in the left panel.
Click on a free area on the canvas to insert an instance of that class.
To draw a Relationclass, select a relationclass.

You are in Drawing Mode Relationclass:

This means you selected a Relationclass in the left panel.
Click on a class instance to start drawing a relationclass instance.
To draw a class, select a class in the left panel.

You are in View Mode

This means you did not select any class or relationclass in the left panel, nor did you click on a class- or relationclass instance.

  • Select a class in the left panel to enter the drawing mode.
  • Select a relationclass in the left panel to enter the drawing relationclass mode.
  • Click on a class- or relationclass instance to enter selection mode.

Create or Load Scene Instance:

  • To create a new scene instance, chose your Scene Type in the left panel and click CREATE NEW SCENEINSTANCE or just double-click on the name of the Scene Type.
  • To load an existing scene instance from the database, chose your scene Type, unfold it by clicking on the arrow button and double-click on your scene instance to load.

Save, Download and Import Scene Instance:

  • To save your scene instance, click on the FILE -> Save Model button in the top menu.
  • To download your scene instance, click on the FILE -> Export Model as .json button in the top menu.
  • To import a scene instance, click on the FILE -> Import Model button in the top menu.

General Actions:

  • Zoom: Use your mouse wheele or touch pad to zoom in and out.
  • Move: Right-click and hold to change the view of the scene.
  • Rotate: Middle-click and hold to rotate the scene. (only in 3D view mode)
  • 2D/3D View Mode: Click on the 2D/3D slider to switch to the view mode.
